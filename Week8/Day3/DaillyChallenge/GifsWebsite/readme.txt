Des Instructions :
Nous allons travailler avec le modèle Gif .

Ajoutez un Integerfield"j'aime" nommé à votre modèle Gif

Ajoutez 2 boutons au modèle de vue Gif :
Bouton vert - lorsqu'il est cliqué, les goûts de ce gif seront incrémentés.
Bouton rouge - lorsqu'il est cliqué, il décrémente les goûts de ce gif.

Créez une page (vue + modèle + chemin) pour afficher tous les gifs qui ont un nombre positif de likes, classez-les sur la page en fonction de leur nombre de likes.
